one-day-game
=======================
[![Build Status](https://travis-ci.org/FreeCX/one-day-game.svg?branch=master)](https://travis-ci.org/FreeCX/one-day-game)

### How to build
- install
    - g++
    - make
    - sdl2
    - sdl2_image
- `make all`
